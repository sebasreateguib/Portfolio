# Portfolio Web

A modern, interactive portfolio website built with **Next.js**, **React**, and **Tailwind CSS**. It features rich animations, a terminal-like interface, and an integrated AI assistant.

## Features

- **Next.js & React**: Fast and modern web framework.
- **Tailwind CSS**: Beautiful, utility-first styling.
- **Framer Motion**: Smooth animations and transitions.
- **AI Integration**: Powered by Google Generative AI (`@google/generative-ai`).
- **Interactive Components**: Includes an ASCII hero section, an interactive globe, and a portfolio terminal interface.
